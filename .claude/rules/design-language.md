# Design Language Rules

## Global
- Dark mode only.
- Grounded motion: avoid floaty easing; avoid excessive parallax.

## DYMO label system
- Mono type labels/buttons, deep inset emboss, subtle rotation (1–2deg max), tactile press state.
- Avoid generic flat buttons.

## Typography hierarchy
- Headings: editorial/serif where appropriate
- Body: clean sans
- Labels/metadata: mono
